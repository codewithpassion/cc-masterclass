# Welcome to Claude Code Masterclass

This is your interactive learning environment for mastering Claude Code, Anthropic's official CLI tool for Claude. Get ready to supercharge your software engineering workflow!

## Prerequisites

Before you begin, you'll need to install Bun, the all-in-one JavaScript runtime and package manager.

### Installing Bun

#### macOS
```bash
curl -fsSL https://bun.sh/install | bash
```

#### Linux
```bash
curl -fsSL https://bun.sh/install | bash
```

#### Windows
Download and run the installer from [bun.sh](https://bun.sh), or use:
```powershell
powershell -Command "(New-Object Net.ServicePointManager).SecurityProtocol = [Net.ServicePointManager]::SecurityProtocol -bor [Net.SecurityProtocolType]::Tls12; irm bun.sh/install.ps1 | iex"
```

After installation, verify it works:
```bash
bun --version
```

## Setting Up the Marketplace

Install the Claude Code marketplace and all plugins with a single command:

```bash
curl -fsSL https://raw.githubusercontent.com/codewithpassion/cwp-claude-marketplace/main/install.sh | bash
```

This will set up all available plugins from the marketplace, giving you instant access to powerful Claude Code extensions.

## Next Steps

Once you have Bun and the marketplace installed, you're ready to explore the masterclass content. Happy coding!
